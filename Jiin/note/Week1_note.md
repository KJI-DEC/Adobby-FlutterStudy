# 1주차(1-8강)

### **설치 및 환경변수 설정**

**설치**

- flutter.dev에서 zip 파일 다운로드 및 압축해제
- Android Studio 설치 및 emulator 설치

**환경변수 설정**

- 시스템변수 - Path에서 새로만들기를 눌러 flutter\bin 파일의 위치를 넣어줌
- cmd에서 flutter 입력 시 잘 작동되는 것을 확인할 수 있음
